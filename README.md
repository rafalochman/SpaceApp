# NASAapp
NASAapp is a web aplication which uses NASA WebApi to show daily photos with description and show asteroids which miss Earth with their history. 

## Technologies
* C#
* ASP.NET Core MVC
* .NET Framework 5.0
* HTML, CSS, JavaScript
* Bootstrap
* WebApi
